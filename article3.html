课程概述：零基础入门 Python 只需 3 步

我们相信，Learning by doing 是最高效的学习方法。
因此，课程中设计了大量实战项目。课程中的视频和示例练习，是为了帮助你掌握完成项目设计所需要技能。看完视频并不代表学习的完成，只是一个开始，在规定的时间内独立完成所有实战项目才可获得硅谷认证的毕业证书。

实战项目能够帮助快速掌握包括 Python 在内的编程语言，原因是：

在完成实战项目的过程中，你会具体应用到所学知识，在应用的过程中夯实、巩固，把知识变成技能。 在实战项目中，你会面对一个实际问题，你需要利用你已掌握的知识，结合自己寻找解决方案的能力。这项技能，也只能在解决具体问题时得到锻炼。 所以，Udacity 不会单方面灌输知识、更不会手把手教你敲代码，而是通过实战项目培养你独立解决问题的能力，并且在这个过程中把知识变成你真正的代码能力。这项能力会陪伴你一生，让你前途无限，成为高水平的抢手人才！
